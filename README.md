# Canvas City Premium

Премиальный сайт-портфолио на реальных фотографиях из загруженных архивов.

## Размещение на GitHub Pages
1. Удалите старые файлы из репозитория `maker-soft/Fon`.
2. Загрузите `index.html` и папку `assets` целиком.
3. Нажмите Commit changes.
4. Подождите обновления GitHub Pages.

Все внешние переходы ведут только в VK: https://vk.com/canvas_city_nsk
