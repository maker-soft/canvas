# Canvas City — сайт для GitHub Pages

Сайт переработан под сообщество:
https://vk.com/canvas_city_nsk

## Изменения
- бренд Canvas City;
- убраны цены;
- убраны Telegram, MAX и другие социальные сети;
- все кнопки, карточки и контакты ведут только в VK;
- добавлены локальные изображения фотофонов;
- отражены материалы и направления использования;
- адаптация для iPhone, планшетов и компьютеров.

## Размещение
Загрузите `index.html` и папку `assets` в корень репозитория `maker-soft/Fon`.
Важно загрузить папку `assets` целиком.

После Commit changes подождите 1–3 минуты и откройте сайт в приватной вкладке Safari.
